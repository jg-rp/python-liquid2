Submodules#

liquid2.builtin.filters.array module#

Filter functions that operate on arrays.

liquid2.builtin.filters.array.compact(sequence: Sequence[Any], key: object = None) → list[object]#

Return a copy of _sequence_ with any NULL values removed.

liquid2.builtin.filters.array.concat(sequence: Sequence[object], other: Sequence[object]) → list[object]#

Return the concatenation of _sequence_ and _second_array_.

liquid2.builtin.filters.array.first(obj: Any) → object#

Return the first item of collection _obj_.

liquid2.builtin.filters.array.join(sequence: Iterable[object], separator: object = ' ', *, environment: Environment) → str#

Return a string by joining items in _sequence_, separated by _separator_.

liquid2.builtin.filters.array.last(obj: Sequence[Any]) → object#

Return the last item of array-like object _obj_.

liquid2.builtin.filters.array.map_(sequence: Sequence[object], key: object) → list[object]#

Return an array/list of items in _sequence_ selected by _key_.

liquid2.builtin.filters.array.reverse(array: Sequence[object]) → list[object]#

Reverses the order of the items in an array.

liquid2.builtin.filters.array.sort(sequence: Sequence[Any], key: object = None) → list[object]#

Return a copy of _sequence_ in ascending order.

When a key string is provided, objects without the key property will be at the end of the output list/array.

liquid2.builtin.filters.array.sort_natural(sequence: Sequence[object], key: object = None) → list[object]#

Return a copy of _sequence_ in ascending order, with case-insensitive comparison.

When a key string is provided, objects without the key property will be at the end of the output list/array.

liquid2.builtin.filters.array.sum_(sequence: Sequence[object], key: object = None) → float | int | Decimal#

Return the sum of all numeric elements in _sequence_.

If _key_ is given, it is assumed that sequence items are mapping-like, and the values at _item[key]_ will be summed instead.

liquid2.builtin.filters.array.uniq(sequence: Sequence[Any], key: object = None) → list[object]#

Return a copy of _sequence_ with duplicate elements removed.

liquid2.builtin.filters.array.where(sequence: Sequence[object], attr: object, value: object = None) → list[object]#

Return a list of items from _sequence_ where _attr_ equals _value_.

liquid2.builtin.filters.math module#

Maths related filter functions.

liquid2.builtin.filters.math.abs_(left: float | int) → float | int#

Return the absolute value of number _num_.

liquid2.builtin.filters.math.at_least(left: float | int, arg: float | int) → float | int#

Return _val_ or _other_, whichever is greater.

liquid2.builtin.filters.math.at_most(left: float | int, arg: float | int) → float | int#

Return _val_ or _other_, whichever is smaller.

liquid2.builtin.filters.math.ceil(left: float | int) → float | int#

Return _num_ rounded up to the next integer.

liquid2.builtin.filters.math.divided_by(left: float | int, right: object) → float | int#

Return the result of dividing _num_ by _other_.

If both _num_ and _other_ are integers, integer division is performed.

liquid2.builtin.filters.math.floor(left: float | int) → float | int#

Return _num_ rounded down to the next integer.

liquid2.builtin.filters.math.minus(left: float | int, right: float | int) → float | int#

Return the result of subtracting _other_ from _num_.

liquid2.builtin.filters.math.modulo(left: float | int, right: float | int) → float | int#

Return the remainder of dividing _num_ by _other_.

liquid2.builtin.filters.math.plus(left: float | int, right: float | int) → float | int#

Return the result of adding _other_ to _num_.

liquid2.builtin.filters.math.round_(left: float | int, digits: int | None = None) → float | int#

Returns the result of rounding _num_ to _digits_ decimal digits.

liquid2.builtin.filters.math.times(left: float | int, right: float | int) → float | int#

Return the result of multiplying _num_ by _other_.

liquid2.builtin.filters.misc module#

Miscellaneous filter functions.

liquid2.builtin.filters.misc.date(dat: datetime.datetime | str | int, fmt: str, *, environment: Environment) → str#

Return a string representation of _dat_ using format string _fmt_.

liquid2.builtin.filters.misc.default(obj: Any, default_: object = '', *, allow_false: bool = False) → Any#

Return _obj_, or _default_ if _obj_ is nil, false, or empty.

liquid2.builtin.filters.misc.size(obj: Any) → int#

Return the length of _obj_.

_obj_ could be a dict, list, string or any class implementing _len_.

liquid2.builtin.filters.string module#

Filter functions that operate on strings.

liquid2.builtin.filters.string.append(val: str, arg: object) → str#

Return a copy of _val_ concatenated with _arg_.

If _arg_ is not a string, it will be converted to one before concatenation.

liquid2.builtin.filters.string.capitalize(val: str) → str#

Return _val_ with the first character in uppercase and the rest lowercase.

liquid2.builtin.filters.string.downcase(val: str) → str#

Return a copy of _val_ with all characters converted to lowercase.

liquid2.builtin.filters.string.escape(val: str, *, environment: Environment) → str#

Return _val_ with the characters &, < and > converted to HTML-safe sequences.

liquid2.builtin.filters.string.escape_once(val: str, *, environment: Environment) → str#

Return _val_ with the characters &, < and > converted to HTML-safe sequences.

It is safe to use escape_one on string values that already contain HTML escape sequences.

liquid2.builtin.filters.string.lstrip(val: str) → str#

Return a copy of _val_ with leading whitespace removed.

liquid2.builtin.filters.string.newline_to_br(val: str, *, environment: Environment) → str#

Return a copy of _val_ with LF or CRLF converted to <br />, plus a newline.

liquid2.builtin.filters.string.prepend(val: str, arg: str) → str#

Return a copy of _arg_ concatenated with _val_.

liquid2.builtin.filters.string.remove(val: str, arg: str) → str#

Return a copy of _val_ with all occurrences of _arg_ removed.

liquid2.builtin.filters.string.remove_first(val: str, arg: str) → str#

Return a copy of _val_ with the first occurrence of _arg_ removed.

liquid2.builtin.filters.string.remove_last(val: str, arg: str) → str#

Return a copy of _val_ with last occurrence of _arg_ removed.

liquid2.builtin.filters.string.replace(val: str, seq: str, sub: str = '') → str#

Return a copy of _val_ with each occurrence of _seq_ replaced with _sub_.

liquid2.builtin.filters.string.replace_first(val: str, seq: str, sub: str = '') → str#

Return a copy of _val_ with the first occurrence of _seq_ replaced with _sub_.

liquid2.builtin.filters.string.replace_last(val: str, seq: str, sub: str) → str#

Return a copy of _val_ with the last occurrence of _seq_ replaced with _sub_.

liquid2.builtin.filters.string.rstrip(val: str) → str#

Return a copy of _val_ with trailing whitespace removed.

liquid2.builtin.filters.string.safe(val: str, *, environment: Environment) → str#

Return a copy of _val_ that will not be automatically HTML escaped on output.

liquid2.builtin.filters.string.slice_(val: Any, start: Any, length: Any = 1) → str | list[object]#

Return the subsequence of _val_ starting at _start_ with up to _length_ chars.

Array-like objects return a list, strings return a substring, all other objects are cast to a string before returning a substring.

liquid2.builtin.filters.string.split(val: str, sep: str) → list[str]#

Split string _val_ on delimiter _sep_.

If _sep_ is empty or _undefined_, _val_ is split into a list of single characters. If _val_ is empty or equal to _sep_, an empty list is returned.

liquid2.builtin.filters.string.strip(val: str) → str#

Return a copy of _val_ with leading and trailing whitespace removed.

liquid2.builtin.filters.string.strip_html(val: str, *, environment: Environment) → str#

Return a copy of _val_ with all HTML tags removed.

liquid2.builtin.filters.string.strip_newlines(val: str, *, environment: Environment) → str#

Return ta copy of _val_ with all newline characters removed.

liquid2.builtin.filters.string.truncate(val: str, num: Any = 50, end: str = '...') → str#

Return a copy of _val_ truncated to _num_ characters.

liquid2.builtin.filters.string.truncatewords(val: str, num: Any = 15, end: str = '...') → str#

Return a copy of _val_ truncated to at most _num_ words.

liquid2.builtin.filters.string.upcase(val: str) → str#

Return a copy of _val_ with all characters converted to uppercase.

liquid2.builtin.filters.string.url_decode(val: str) → str#

Return a copy of _val_ after decoding percent-encoded sequences.

liquid2.builtin.filters.string.url_encode(val: str, *, environment: Environment) → str#

Return a percent-encoded copy of _val_ so it is useable in a URL.
